The SLG46621 GreenPAK4 is Silego’s fourth generation GreenPAK Product. It is a one-time programmable Mixed signal array available in a small 2mm x 3mm TDFN-20 package.

Features

• Logic & mixed signal circuits;

• Highly versatile macro cells;

• Read back protection (Read lock);

• 1.8 V (±5%) to 5 V (±10%) supply;

• 1.8 V (±5%) to 3.3 V (±10%) supply (when using the 2MHz internal OSC);

• Operating temperature range: -40°C to 85°C;

• RoHS compliant / Halogen-free;

• 20-pin STQFN: 2 × 3 × 0.55 mm, 0.4 mm pitch;


Description

The SLG46621 provides a small, low power component for commonly used mixed-signal functions. The user creates their circuit design by programming the one time Non-Volatile Memory (NVM) to configure the interconnect logic, the I/O Pins and the macro cells of the SLG46621. This highly versatile device allows a wide variety of mixed-signal functions to be designed within a very small, low power single integrated circuit. The macro cells in the device include the following (please see Block Diagram below):

• Six Analog Comparators (ACMP);

• Voltage References (Vref) with Chopper stabilized amplifier;

• Twenty five combinatorial Look Up Tables (LUTs);

- Eight 2-bit LUTs;

- Sixteen 3-bit LUTs;

- One 4-bit LUT;

- One combination function macrocell;

• One selectable Pattern Generator (PGEN) or 4-bit LUT;

• Ten Counter / Delay generators (CNT/DLY);

- Four 14-bit Delay/Counters;

- Six 8-bit Delays/Counters;

• Two Pipe Delays – 16 stage/3 output (Part of combination function macrocell);

• Two Programmable Delays;

• Additional logic functions – 2 Inverters;

• RC Oscillator (RC OSC);

• Low Frequency Oscillator (LF OSC);

• Ring Oscillator (Ring OSC);
