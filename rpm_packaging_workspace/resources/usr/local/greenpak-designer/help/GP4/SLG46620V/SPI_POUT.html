The SPI Parallel Output is a way to fan out bits [7:0] of the SPI to the matrix. Available only in matrix 1. Parallel output bits 0 through 7 connections are muxed with DCMP1, DMCP2 outputs and LF OSC CLK, OUT1, OUT0 OSC outputs. Choose which outputs to use based on the enable signal of the SPI Parallel output block.

Activation

Disconnect the SPI block’s nCSB from VDD and select ‘Enable’ under SPI Parallel Output’s properties.

Parameters

SPI Parallel Output:

Enable/Disable SPI Parallel Output
