The Bandgap Good (BG) macrocell provides an output that is triggered by the bandgap status. BG settings can be found under ADC, Vref and ACMP register settings.

Parameters

Output delay time:

A selectable delay is added after the bandgap voltage has become valid. This gives analog blocks time to start up before valid data is acquired.

• 100us

• 550us: select if charge pump is used to allow for charge pump startup time;


Force bandgap on:

Set control of bandgap power-on.

• Disable: Automatically turn the bandgap on and off based on analog block usage.

• Enable: Force bandgap on.
