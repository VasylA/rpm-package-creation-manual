Lock status:

This option allows the locking of NVM code in the programmed chip, preventing a subsequent read of the chip NVM sequence.

Pattern ID:

An 8-bit code programmed onto the chip that can be used to identify the NVM code programmed onto the chip.

Power Supply Control mode:

• Regulator always ON and Charge Pump is automatically ON/OFF (use for dynamic 1.71V < VDD < 5.5V): Charge pump uses auto power detector function. Auto power detector function auto turns on charge pump when VDD is detected <2.7V. When VDD is >2.7V, the charge pump will be off.

• Regulator always ON and Charge Pump always OFF (use for dynamic VDD > 2.7V): Charge pump auto power detector function is off therefore Charge pump is always off. at VDD < 2.7V, Analog components may not work as expected.


GPIO quick charge:

During the POR sequence, a 2kΩ resistor is connected in parallel to any configured pull-up/down resistors. The 10K, 100K and 1M GPIO Pull-Up/Pull-Down resistors are not enabled until POR so the quick-charge option can help inputs get to the right voltage faster, especially if there is significant capacitance.
