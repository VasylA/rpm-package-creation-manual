VDD2 is the second power supply which powers the input/output buffers for pins, dedicated to VDD2 voltage domain.

Parameters

Min., Typ., Max.:

Set of operating conditions. These settings are for the user only. They do not affect the NVM bits but affect the VDD2 range limit the chip will work at. See datasheet for electrical characteristics.
These settings can be set by changing specs values (“Project Info” dialog).
