The SLG46580 GreenPAK5 is Silego’s first generation Programmable Power Management IC. It is a one-time programmable Mixed signal array available in a small 2mm x 3mm STQFN package.

Features

• Programmable Asynchronous State Machine;

• Logic and Mixed Signal Circuits;

• Highly Versatile Macro Cells;

• Read Back Protection (Read Lock);

• Four 150mA LDO Regulators;

• I2C Interface;

• 2.3 V (±5%) to 5 V (±10%) Supply;

• Operating Temperature Range: -40°C to 85°C;

• 2 × 3 mm STQFN package;

• RoHS Compliant / Halogen-Free;


Description

The SLG46580 is a small, low power component for commonly used to integrate mixed-signal functions under control of an asynchronous state machine. The user creates their circuit design by programming the one time Non-Volatile Memory (NVM) to configure the interconnect logic, the I/O Pins and the macro cells of the SLG46580. This highly versatile device allows a wide variety of functions and control logic to be designed within a very small, low power single integrated circuit. The macro cells in the device include the following:

• Four Analog Comparators (ACMP);

• Voltage Reference (Vref) for ACMPs;

• Analog Temperature Sensor;

• Fifteen Combination Function Macrocells;

- Three Selectable DFF/Latch or 2-bit LUTs;

- Six Selectable DFF/Latch or 3-bit LUTs;

- One Selectable Pipe Delay or 3-bit LUT or Ripple Counter or 3-bit LUT;

- Five 8-bit delays/counters or 3-bit LUTs;

• Combinatorial Logic;

- One 4-bit LUT with two outputs;

• Asynchronous State Machine;

- Eight States;

- Flexible input logic from state transitions;

• Real Time Clock (RTC) Binary Counter;

• LDO Regulators;

- Four Dual Mode 150mA LDO Regulators with Quiescent Current at 60uA for Mode 0, and 6 uA for Mode 1;

• Serial Communications;

- Silego’s Proprietary 2-wire interface “PC2W”;

- I2C Slave Protocol Interface;

• Programmable Delay with Edge Detector Output;

• Additional logic functions

- 2 Deglitch Filters with Edge Detectors;

• Two Oscillators (OSC);

- Configurable 25 kHz/2 MHz;

- 2kHz Low Power Oscillator;

• Eight Byte RAM + OTP User Memory;

- RAM Memory space that is readable and writable via I2C;

- User defined initial values transferred from OTP;

• POR
