The SLG46580V comes with four low dropout regulators each rated at 150 mA. Each LDO regulator has 3 modes which are: HP MODE is the standard active mode; LP MODE is a low power mode performance compromised mode; and finally BYPASS MODE in which the LDO regulator ceases to regulate and the Regulator MOSFET is turned on as a power switch passing the voltage applied to VIN directly to VOUT.

The LDO regulators are paired together with LDO1 and LDO2 sharing the same VIN called LDO_Vin0, likewise LDO3 and LDO4 share the same VIN called LDO_Vin1.

Activation

Connect the ‘ENABLE’ input to a logic high signal.

Parameters

Mode:

The MOSFET gate is regulated by a comparator. Chose which input to use as the Vref.

VOUT0 voltage:

In increments of 100mV. 0.9V, 1.0V … 4.20, 4.30, 4.35V.

VOUT1 voltage:

In increments of 100mV. 0.9V, 1.0V … 4.20, 4.30, 4.35V.

Start-up ramping slope:

A slow start-up ramp when the LDO is enabled. Either 10V/ms or 20V/ms.

Discharge resistor:

If the LDO is disabled, a 300 pull down can be asserted on LDO’s Vout to discharge.

Overcurrent & short-circuit detection:

Enable Over-current and Short-Circuit-Current protection.

Connections

UVLO:

This parameter is shared with the corresponding ACMP properties. If Enabled, the ACMP output will be routed to the UVLO input of the LDO.
