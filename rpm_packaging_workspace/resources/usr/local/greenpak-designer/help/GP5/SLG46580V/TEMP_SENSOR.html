The temperature sensor (TEMP SENSOR) takes the temperature sensitive node of the bandgap and outputs to ACMP2’s IN+ source.

If Enabled, the bandgap and output active buffer are forced on. For more information, see VREF0 Help Info.

Activation

Set the parameter ‘Enable temp. sensor’ to Enable. ACMP2’s IN+ source must be selected as Temp Sensor.

Parameters

Enable temp. sensor:

Enable or Disable this macro-cell.

Output:

The output can be routed to ACMP2’s IN+ source.
