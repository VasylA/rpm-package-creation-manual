The Ripple Counter is a 3-bit counter that rolls over once it reaches its End Value (EV). It has three inputs from the matrix: UP, nSET, and CK. The block has 3 matrix outputs.

Description

Inputs:

• UP: Determines the direction of the Ripple Counter. When UP is HIGH, the Ripple Counter will count up. When UP is LOW, the Ripple Counter will count down.

• nSET: For correct Ripple Counter functionality nReset (nRST) input should be HIGH. If nReset (nRST) input is LOW the Ripple Counter is in a reset state and the outputs return to the Set Value (SV).

• CK: Clocks the Ripple Counter.


Outputs:

The Ripple Counter has three outputs to the matrix, which correspond to its count value: Q2, Q1, Q0.

Parameters

nSET value (SV):

Choose the Set Value for the Ripple Counter.

End value (EV):

Choose the End Value for the Ripple Counter.

Functionality mode:

Choose whether the Ripple Counter counts from 0 to the End Value, or from the Set Value (SV) to the End Value.
